# Agent Notes

This folder stores task-level handoff notes from coding agents.

Each task that modifies the repository should create one Markdown file using this naming pattern:

`YYYY-MM-DD_<short_task_slug>.md`

These notes help later agents understand what changed, what was validated, what failed, and how to continue work.

Do not store raw controlled data, credentials, API keys, sensitive row-level records, or copied source documents in this folder.
